The Living Seminar · 2026

Von Neumann × Sun Tzu

A playable seminar on formal games, strategic terrain, deception, mechanism design, and multi-agent futures.

What changed? In 1945, this encounter could only be imagined. Today it can be staged as an interactive intellectual machine: one voice formalises conflict as game structure; the other treats strategy as terrain, perception, timing, and asymmetry. This fixed version runs immediately as a self-contained demo, and can be connected to a secure LLM proxy when you want the voices to be truly model-generated.
Default mode: fully working browser demo. No API key, no server, safe for GitHub Pages / Netlify static hosting. True LLM mode: add a private proxy endpoint below; never place an Anthropic API key directly inside public HTML.

What Von Neumann contributes
Payoffs, equilibria, commitment

What Sun Tzu contributes
Terrain, timing, perception
The strategic voice asks whether the game itself can be reshaped before battle: shift terrain, alter incentives, confuse observation, or win by not entering the expected contest.
What the exercise reveals
Mechanism design as battlefield design
The synthesis usually emerges when equilibrium selection meets terrain design: do not merely choose a move; change the rules under which moves become rational.
